# Rermote BLE controller for handlebar on bikes or motorbikes

## introduction

There is an single ESP32-C3 controller (RISC-V) and an analog joystick and a few keys

The controller builds an combo device, a mouse device and an keyboard device at once. This is an 
standart HID Device via blue thooth low energy, an well known standart in it.

## goodies

you can use this device i.e. on an motorbike handlebar to control your moblile device. 
I use it for navigation with navigation software like the popular DMD2.
the device support energy save techniques, it meand when no activities can the device switch 
into one simpll sleep state or later in a depp sleep state.
